# 0xlightning Portfolio

Personal portfolio website for **Praveen KS (0xlightning)**, a Full Stack Developer specializing in SaaS, eCommerce, and real-time web applications.

## 🚀 Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router, Static Export)
- **Styling:** [Tailwind CSS](https://tailwindcss.com) v4
- **Components:** [shadcn/ui](https://ui.shadcn.com/)
- **Animations:** [GSAP](https://gsap.com/) & [Lenis](https://studio-freight.github.io/lenis/) (Smooth Scrolling)
- **Typography:** Inter & Fira Code (Google Fonts)

## 🛠️ Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Deployment

This project is configured to automatically deploy to **GitHub Pages** using GitHub Actions.

1. Ensure your repository has GitHub Pages enabled.
2. Set the source to **GitHub Actions**.
3. Push to the `main` branch to trigger a deployment.

The deployment workflow is defined in `.github/workflows/nextjs.yml`.

## 📈 SEO & Structured Data

The portfolio is fully optimized for SEO, AEO, and GEO out-of-the-box, featuring:
