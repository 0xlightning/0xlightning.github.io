# Skills Section

## Recommended Style

Instead of basic skill bars:

Use:

* Modern skill cards
* Floating stack badges
* Interactive hover glow
* Animated reveal on scroll

### Example Categories

* Frontend
* Backend
* Shopify
* CMS
* Motion
* UI/UX
* Performance
