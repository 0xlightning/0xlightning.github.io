### Effects

* React Three Fiber (light use only)
* CSS blend modes
* SVG path animations

## Smooth Section Transitions

### Recommended Effects

#### Between Sections
* Fade + translate
* Blur-in transitions
* Scroll-linked opacity
* Gradient fade dividers

#### Avoid
* Overly dramatic transitions
* Excessive 3D movement
* Heavy particle systems

## Micro Interactions

### Buttons

* Magnetic hover effect

### Cards

* Floating motion
