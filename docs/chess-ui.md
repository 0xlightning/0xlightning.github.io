# Chess-Themed UI Integration

## Important Rule

The chess theme should feel:

* Elegant
* Strategic
* Artistic
* Minimal

NOT:

* Gaming-heavy
* Over-decorated
* Cartoonish

## Floating Chess Pieces

### Placement Strategy

Use low-opacity SVG chess pieces:

* Left edge of hero
* Right side of sections
* Background corners
* Timeline edges
* Footer background

### Recommended Pieces

* King
* Queen
* Knight
* Bishop

Avoid using too many pawns.

### Motion Style

* Slow floating animation
* Slight rotation
* Parallax movement
* Blur on background layers
* Opacity around 0.03–0.08
