# Visual Theme

## Main Direction

* White / off-white background
* Light gray layered sections
* Soft shadows with subtle elevation
* Minimal black typography
* Neutral accent palette
* Clean spacing with breathing room
* High-end modern developer aesthetic

## Optional Accent Colors

Use very lightly:

* Slate blue
* Soft silver
* Muted beige
* Matte graphite

Avoid strong neon colors.
