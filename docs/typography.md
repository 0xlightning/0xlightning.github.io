# Typography Hierarchy

## Recommended Fonts

### Main Font

* Inter
* Satoshi
* General Sans
* Plus Jakarta Sans

### Accent Font

Use lightly for headings:

* Clash Display
* Cabinet Grotesk
* Neue Montreal

## Typography Rules

Hero Heading:

```css
font-size: clamp(3rem, 8vw, 6rem);
font-weight: 700;
line-height: 0.95;
letter-spacing: -0.04em;
```

Body:

```css
font-size: 16px;
line-height: 1.8;
color: #666;
```
