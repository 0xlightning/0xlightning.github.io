# Navbar Improvements

## Recommended Navbar

### Structure

Left:

* Logo / initials

Center:

* Navigation links

Right:

* Theme toggle
* Resume button
* Chess icon micro-animation

### Style

* Glassmorphism light blur
* Thin border
* Floating rounded navbar
* Sticky on scroll

### Hover Effects

* Smooth underline animation
* Glow blur hover
* Slight translateY effect
